| Property | Values |
|----------|--------|
| `fontFamily` | `Noto Sans JP` (required on all text) |
| `fontSize` | 12 (small), 14 (normal), 18 (recommended), 24 (title) |
| `fontColor` | `#RRGGBB` (default: black) |
| `fontStyle` | 0=normal, 1=bold, 2=italic, 4=underline (additive: 3=bold+italic) |
| `align` | left, center, right |
| `verticalAlign` | top, middle, bottom |
