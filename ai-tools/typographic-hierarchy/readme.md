# Typographic Hierarchy

**Type:** skill | **Version:** 1.0.0 | **OS:** any

Format documentation, READMEs, and structured text using header hierarchy where each level stands alone. Makes content scannable at multiple reading depths.

## Tags
typography, formatting, documentation, readability, hierarchy

## Overview
This skill applies typographic hierarchy principles to structure text so that headers tell the complete story on their own. Content is organized for three reading depths: glancers understand the concept from top-level headers, skimmers get key takeaways from mid-level headers, and readers access full supporting details. The approach works across technical docs, marketing copy, research notes, and any medium that benefits from clear visual information hierarchy.
