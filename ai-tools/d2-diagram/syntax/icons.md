# Icons

- Add icon URL: `icon: https://icons.terrastruct.com/...`
- `shape: image` makes the node icon-only (no box)
- Icons appear inside the node alongside the label

```d2
server: API Server {
  icon: https://icons.terrastruct.com/tech%2F022-server.svg
}

cloud: {
  icon: https://icons.terrastruct.com/aws%2F_Group%20Icons%2FAWS-Cloud-alt_light-bg.svg
  shape: image
}
```
