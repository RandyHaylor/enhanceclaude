# Styles

All styles live under `.style.*` on any node or connection.

```d2
node.style.fill: "#e3f2fd"
node.style.stroke: "#1565c0"
node.style.stroke-width: 2
node.style.stroke-dash: 3       # dashed border
node.style.border-radius: 8     # rounded corners
node.style.font-color: "#333"
node.style.font-size: 14
node.style.bold: true
node.style.italic: true
node.style.opacity: 0.5
node.style.shadow: true
node.style.multiple: true       # stacked visual (for db clusters)
node.style.3d: true             # 3D box

# connections
edge.style.stroke-dash: 5
edge.style.animated: true
```
