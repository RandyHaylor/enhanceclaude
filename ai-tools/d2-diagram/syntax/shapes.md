# Shapes

Set with `node.shape: <value>` or inline `node: { shape: cylinder }`

| Shape | Use for |
|---|---|
| `rectangle` | default, generic step |
| `oval` | start / end |
| `diamond` | decision / branch |
| `cylinder` | database, cache |
| `hexagon` | gateway, load balancer |
| `person` | user, actor |
| `queue` | message queue |
| `cloud` | external service, internet |
| `document` | file, report |
| `stored_data` | data store |
| `parallelogram` | I/O operation |
| `page` | document/page |
| `callout` | annotation |
| `circle` | event |
| `package` | module/package |
| `class` | UML class |
| `sql_table` | DB table (special) |
| `sequence_diagram` | sequence (special) |
| `image` | icon/image (needs `.icon: url`) |
