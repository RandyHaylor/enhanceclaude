# Fine-Grained Skill Builder

**Type:** skill | **Version:** 1.0.0 | **OS:** any

Guide for building fine-grained skills where content is split into many small files to minimize tokens loaded per invocation.

## Tags
skill-building, token-optimization, development

## Overview
This skill provides a structured approach to building token-efficient skills using section-based progressive disclosure. Content is broken into small, purpose-specific files that are loaded only when needed, minimizing tokens per invocation. It includes rules for structuring SKILL.md files, section indexes, and content files, along with a methodology for parallelizing skill creation across agent teams.
