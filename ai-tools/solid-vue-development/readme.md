# SOLID Vue Development

**Type:** skill | **Version:** 1.0.0 | **OS:** any

Core principles for writing clean, disciplined Vue 3 apps. Covers state vs behavior separation, component boundaries, manager pattern, reactive chains, and naming conventions.

## Overview
SOLID Vue Development provides a set of core principles for building maintainable Vue 3 applications. It enforces clear separation between state and behavior, defines strict component boundaries where components call functions rather than owning logic, and promotes the manager pattern for encapsulating domain state and behavior. The skill also covers proper use of reactive chains, explicit naming conventions, and scope discipline to prevent premature abstraction.
