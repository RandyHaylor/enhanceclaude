# RTF Creator

**Type:** skill | **Version:** 1.0.0 | **Platform:** Claude Code

Generate valid RTF (Rich Text Format) documents from scratch. Produces formatted documents compatible with Word, LibreOffice, and other word processors.

## Tags
rtf, document-creation, formatting, word-processing

## Overview
This skill generates structurally valid RTF documents with full support for character formatting, paragraph styling, tables, headers/footers, hyperlinks, and page setup. It includes comprehensive reference material for RTF control words and a built-in validation step using rtfparse to catch unbalanced braces and structural errors before delivery.
