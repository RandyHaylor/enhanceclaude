# RTF Creator

**Type:** skill | **Version:** 1.0.0 | **OS:** any

Generate valid RTF (Rich Text Format) documents from scratch. Produces formatted documents compatible with Word, LibreOffice, and other word processors.

## Tags
rtf, document-creation, formatting, word-processing

## Overview
This skill generates structurally valid RTF documents with full support for character formatting, paragraph styling, tables, headers/footers, hyperlinks, and page setup. It includes comprehensive reference material for RTF control words and a built-in validation step using rtfparse to catch unbalanced braces and structural errors before delivery.

## Try These Prompts
- Create an RTF document with a title, two sections, and a table of data
- Convert this markdown content into a formatted RTF file I can open in Word
- Generate an RTF report with a header, footer with page numbers, and bold section headings
- Make an RTF memo with the company name centered at the top and body text in Times New Roman 12pt

## Use Cases
- Generating formatted reports for Word or LibreOffice
- Creating templated business documents (memos, letters, contracts)
- Producing RTF output from structured data or AI-generated content
- Building documents with tables, headers, and custom fonts without a word processor
- Converting plain text or markdown to a shareable formatted file

## Additional Requirements
Optional: pip install rtfparse for validation.
